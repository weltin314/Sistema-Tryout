# 🎮 Sistema Tryout

Sistema e aplicação web desenvolvido para gerenciamento e avaliação de jogadores de League of Legends em processos de tryout.

---

## 📌 Sobre o projeto

O **Sistema Tryout** tem como objetivo ajudar equipes a organizarem e avaliarem jogadores durante peneiras (tryouts), permitindo anotações, avaliações e análise de desempenho.

---

## 🚀 Funcionalidades

- 📋 Cadastro de jogadores
- ⭐ Sistema de avaliação
- 📝 Anotações individuais
- 📊 Organização de informações
- 🔍 Visualização dos dados
